## Easy installation

`dynami` is available as a Python package on PyPI.org and can be installed with the following command:

```shell copy
python3 -m pip install dynami
python3 -m dynami ensurepath
```

*Depending on the OS, the installation may require additional parameters.*

## Manual installation

For system administrators and the use in companies we recommend the manual installation way.
