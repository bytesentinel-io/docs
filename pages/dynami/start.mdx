# Quick Start

## First setup

Since `dynami` is Python based, `dynami` can be installed on any common operating system.

<Steps>

### Installation

Python must be installed for the installation. `dynami` does not require a Python version, but we always recommend to install the latest version.

<Tabs items={["Windows", "Linux", "macOS"]}>
    <Tab>
        ```shell
        pip install dynami
        ```
    </Tab>
    <Tab>
        ```shell
        pip install dynami
        ```
    </Tab>
    <Tab>
        ```shell
        pip install --user dynami
        ```
    </Tab>
</Tabs>

</Steps>
